
#dev-laravel-develop-example

Laravel 框架可开发版本

一、目的

为了分享学习Laravel框架底层逻辑，并进行实际应用。<br/>
下载下来可以直接应用于正式项目开发使用。

二、关于版本及主要内容说明

1、基于Laravel6.20.1 官方框架包封装，主版本：V1.0-6.20；<br/>
2、提供了web、api、h5多操作端共用一个框架包的后端架构；<br/>
3、翻译了核心源码，对源码逻辑未作任何改动；<br/>
4、封装或改写了一些常用的组件；<br/>
5、提供了一些实际应用的demo；<br/>
6、PHP最低版本要求：7.2。<br/>
